Bye 2018, Hello 2019

发表于 2018-12-30   |   更新于 2018-12-30 | 分类于 随笔 | | 阅读次数:
字数统计: 3,095 字 | 阅读时长 ≈ 10 分钟

第一次写自己写年终总结，突然不知道该咋写。记得以前高中的时候，好像有要求写周记，当时自己也没有认真对待过，都是随便写一些乱七八糟的东西上去。现在想想，坚持写一写的话，没准现在还能当作个回忆，笑一笑当时的自己是多么的稚嫩，多么的天真。

想一想，还是从最近的找实习这个事情开始吧。现在是大三上学期，大概11月中旬的时候，辅导员在年级的官方通知群发了一条“16级前端实习交流群的。。。”的消息，这个消息一出来，顿时感觉自己有点方。原来大家都已经在着手准备实习的事情了，虽然我也算是开始准备了，但是并没有那么认真的去准备。后来做后端的同学们也建立了后端实习交流群，里面已经有一些大佬在面试了，也有他们记录的面试题目，看了看那些面试里问到的题目，真的是一点都看不懂，这个时候才意识到原来自己跑偏了，以前一直认为自己会写代码，写过很多代码，到了找工作的时候就一定能找到。其实公司对大家的要求还是蛮高的，不仅要求会写代码，还要理解背后的原理。还好看到的时候还不晚，还有时间准备。就这样开始了苦逼的复习，好在辛苦没有白费，成功找到了实习。

简历也投了几家公司，刚开始也挺惨的。有个同学在大三上学期已经有实习了，然后找他帮忙内推，很久都没有消息，内心感觉很凉。然后学校联系的企业，我给其中三家公司投了简历，其中包括找同学内推的那家，一家后来有笔试通知，不过笔试直接挂了，自己的基础知识还是不过关，有一些问题就是模棱两可的，拿不准，笔试挂了也在情理之中。还有一家公司，要求相对来说低一点，面试官看我的简历感觉还行，面试过程也挺简单的，比较轻松的就过了。然后自己投了两家公司，第一家是一个上海的公司，话说这家算是我的第一个面试，好多人都说自己在面试的时候会特别紧张，但是我感觉自己还好，刚开始可能会有那么一点，后面慢慢就平静了，第一个面试就聊了大概 20 分钟，然后就没有了后续通知，很显然挂了。然后投了北京的一家公司，也是我大三下学期准备去实习的那家公司，投这家公司的时候大概已经 12 月中旬了，自己一个 offer 还没有，非常的着急，然后在学校的论坛瞎逛，看到了内推的帖子，抱着试试看的心态发了简历邮件。很快就有面试官加了微信约了面试时间，一面面了大概 50 多分钟，面试官人很好，面试过程也很愉快，我有些说全的问题，面试官也会给一些提示能够让我接着说下去，顺利的过了一面。同样的二面加微信约时间，一面问的问题比较基础，二面就感觉有一些深度了，问了一些比较底层的东西，非常庆幸自己有准备，但是一些特别底层的东西还是不会，二面面完，面试官没有给明确的反馈，所以心里也是没有底的。这个时候，最开始找同学帮忙内推的那家公司给了电话面试，电话就简单聊了聊，大概十几分钟的样子，然后约了下午去公司现场面。在去公司的路上，接到了北京那家公司部门主管的电话，给了口头 offer ，当时一直悬着的心感觉终于慢慢落下来了。下午的现场面试也比较随意，就跟普通聊天差不多，差不多确定是给 offer 了。后来同学给我说那天他们的主管只面了我一个人，现在想想没去那家公司还感觉有点不太好意思。

实习面试的事情大概就说这么多吧，反正都写总结了，顺路写一下大学到现在关于学习的一些乱七八糟的东西。我是从大一下学期开学的时候才接触 java 的，当时我们年级的辅导员组建了一个工作室（就是找一些同学做一些项目那种），就是从我们这些大一的学生里找人，当时加入的要求在现在看来挺简单的，用 jsp 写个 “hello world”，会简单的使用 mysql，但是当时是大一下学期刚刚开学，大家的普遍情况是只学过 c 。我当时也对这个工作室有点兴趣，希望可以加入进去学一些东西，现在想想看，这个选择太正确了。当时为了能加入进去，在图书馆泡了两天，上午就去，晚上回宿舍，记得当时配置个环境就把快把自己整懵逼了，还要学习语法什么的，数据库的环境比 java 更恶心，非常庆幸当时没有放弃。进入工作室之后，因为大家都没有什么 web 方面的开发经验，所以主要是学习，前端 html、css 这些，后端主要学习 java。当时给了项目，我们组是做一个成绩发布平台，简单来说就是辅导员从教务系统导出学生成绩 Excel 文件，然后导入到系统内，进行发布，然后学生和家长可以根据学号和姓名来进行查询。当时自己也是太天真，加上辅导员天天催进度，学习的知识一点都不系统，基本上就是各种网上找代码，看看人家怎么写的，然后改成自己的。暑假的时候，整了一本《疯狂java讲义》来看，这么厚的一本书，自己居然能在暑假硬着头皮看了很多，现在想想都有点佩服自己，边看书边做那个项目，暑假算是做出一个雏形吧，但是界面是真的辣眼睛。

暑假开学，就是大二上学期了，辅导员给找了其他的同学来做前端，因为之前小组的同学一直不怎么积极。其实这个时候做的项目还是在瞎做，项目第一版用 jsp/servlet 来做的，当时工作室里没有学长，学习方向上没有人指导，代码写的很烂，本来 jsp 应该用来做视图， servlet用来做处理器，奈何自己当时太菜了，全都用 jsp 做了。大二上学期主要是做页面了，附带后台一些功能的完善，磨磨蹭蹭的也是做了一个学期才做完，寒假的时候试用了一下，问题真的是相当的多。大二学期我们有综合课程设计这个课，大二下，大三上也都有，大二上的选题是一个通讯录系统，我用 servlet 写的后端，那时候才算知道 servlet 这个东西怎么用。大二寒假，因为项目的测试效果差了，就想着改进项目，但是现在也算有经验了，知道先系统的去学一些知识在开始写代码，寒假在家学了 spring 和 mybatis，其实只是简单了解，说不上多么深入。开学之后就是大二下学期，开始讲项目用 SSM 框架来重写。其实看是和写代码还是有很大差距的，你以为自己看会了，等真正写的时候并不是那个样子，当时用了一个星期的时间才把 SSM 框架搭好，成功的把数据库里的东西读出来然后显示在页面上的时候，感觉开心到爆炸。然后用了大概一个月的时间重新设计了数据库，写完了后端。又是到了跟前端结合的艰巨任务，当时还没有意识到写什么 api 文档，也不知道什么前后端分离，所以基本上都是前端写好了发给我，然后在我这边调，坎坎坷坷这一版也算是写完了。后来就到了暑假的实训，学院有要求要进行一个月的实训，当时也是去跟学校有合作的企业，投简历也有点惨，我投的那来那两家公司突然不打算找同学来实训了，直接退了所有人的简历，后来第二批投简历的时候只能去了一家培训，现在看来也不算也别惨。实训的讲师也是从零基础讲起吧，从 jsp/servlet 到 SSM 再到 SpringBoot。我也是在那个时候开始学习的 spirngboot，学了之后才发现这个东西是真的好用，不用再去写那一大堆的配置文件了。实训的项目选题是高校门户网站，采用了 springboot + mybatis 的方式来实现，因为是个实训的项目，所以并没有实际上线运行。

暑假结束就是就是大三了。成绩发布平台那个项目，用了 springboot 来重构，数据库采用了 mysql + mongodb，因为 mysql 毕竟是个关系型数据库，用来存储不确定的 Excel 文件总有些怪怪的，使用 mongodb 就显得非常合理。大三下学期因为要实习的缘故，也开始刷一些算法题了，基本上算是每天一两道题那种吧，后面被一个项目给耽误了。那是帮我老哥的一个朋友做的，主要就是管理家具生产数据的一个系统，顺路记录一下产值之类的信息，因为是帮别人做的项目，并且要真正的应用起来，所以算是做的比较用心的一个项目。这个项目，后面还成了简历上的主力项目，哈哈。然后就是找实习的那一堆事了。

关于找实习的经历，突然觉得多听听前辈们的话，用处还是非常大的。面试完一般都可以对面试官进行提问，让面试官给自己提一些后面学习上的建议真的是一个非常高好的问题。记得北京那家公司的二面，问了很多底层实现方面的东西，有一些问题确实不会，心里没底，最后让面试官给建议。他说，我们不仅要会写代码，更重要的是去理解底层的逻辑，现在你随便找个人都能给你写完业务代码，这些很简单，我们必须要去理解那些底层的东西，因为有时候一些 bug 需要我们理解底层相关的东西才能解决，理解原理才能提升性能等等。所以会搬砖不行，你还要知道为什么这么搬才行。

下学期就要去公司实习了，也说一点对自己的期望。首先在公司踏踏实实的学习，打好基础，提升能力，多向大佬们学习，学习知识，学习思考问题、解决问题的方式等等。多对自己学过的东西进行总结，话说这个博客搭了这么久还没有写过多少东西，有点浪费了。抽时间多读一些书，技术书籍要看，其他书也要尽量去看，提升一下自己这个文学素养，做一个优雅的程序员，毕竟“哇，好多鸟！好多鸟！”和“落霞与孤鹜齐飞，秋水共长天一色”之间，后者明显更好嘛。

好了，啰啰嗦嗦大概就写这么多吧。再见 2018，你好 2019。
