Hexo

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页

将博客部署到GitHub Pages

博客部署可以采用自己的云服务器或者使用 GitHub Pages，因为 GithHub 是免费的，所以推荐使用 GitHub

新建 GitHub 仓库

  • 注册一个 GitHub 账号
  • 新建一个仓库，名称为 你的github账号名称.github.io

  • 博客访问地址为 你的github账号名称（转换为小写）.github.io

将个人域名解析到 GitHub Pages

我是用的是腾讯云的域名，这样就可以通过自己的域名来访问 GitHub Pages 了，例如我的地址为 https://jokeralex.github.io 现在可以通过 http://alexzh.cn 来访问了

  • 打开自己的 GithHub 仓库，打开 Settings 找到 GitHub Pages 将里面的 Custom domian 改成自己的域名，如图所示
